# Domain Redirect Setup

Simple redirect page to https://www.esjfinancial.online/

## Files

- `index.html` - Main HTML page with redirect button

## Usage

1. Open `index.html` in a web browser
2. Click the "Start Free Assessment" button
3. User will be redirected to https://www.esjfinancial.online/

## Deployment

To deploy this:

1. **GitHub Pages**: Upload files to a GitHub repository and enable GitHub Pages
2. **Netlify**: Drag and drop the folder to Netlify
3. **Vercel**: Connect your repository to Vercel
4. **Any Web Host**: Upload `index.html` to your web server

## Customization

- Edit the button text in `index.html`
- Change colors in the `<style>` section
- Modify the redirect URL in the button's `href` attribute
